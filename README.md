# java-cli-maven-failsafe-spring-jbehave-mockito-junit5-car-data

## Description
A POC for maven app using junit5
and jbehave and mockio framework
 with failsafe plugin.

## Tech stack
- java
- maven
	- spring
	- mockito
  - junit5
  - failsafe
  - jbehave

## Docker stack
- adoptopenjdk/maven-openjdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
